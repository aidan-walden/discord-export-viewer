# discord-export-viewer

discord-export-viewer is a Bun / React site written in TypeScript enabling rich viewing of exports from [Discord Chat Exporter](https://github.com/Tyrrrz/DiscordChatExporter) that were exported in JSON format.

Supports Chromium-based browsers **only**.

## Features

Supports:

- [ ] Message viewing
- [ ] Reactions rendering
- [ ] Inline attachment viewing
- [ ] Messages search

## Project Goal

This project intends to:

- Provide a platform-agnostic solution to viewing Discord Chat Exporter exports
- Require no additional downloads from the user to use

Although this project is very similar to [DiscordChatExporter-frontend](DiscordChatExporter-frontend), it aims to differeniate itself in these two ways.
